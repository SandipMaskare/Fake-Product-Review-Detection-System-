
# Fake Product Review Detection System

## Introduction
This project uses Machine Learning and Natural Language Processing to detect fake product reviews.

## Technologies Used
- Python
- Scikit-learn
- Pandas
- Flask (for Web UI)

## Dataset
Includes sample reviews labeled as real (1) or fake (0).

## ML Algorithm
- Naive Bayes Classifier
- CountVectorizer (Text Preprocessing)

## Output
- Accuracy score and confusion matrix
- Trained model saved as .pkl

## Future Scope
- Larger dataset for better accuracy
- Integration into real e-commerce platforms
